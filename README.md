# MakeCode Tutorials

## 🎮 Game Building Sessions

Fast-paced 1:1 tryout sessions. Students build a complete, playable game with a Sensei in under 30 minutes.

- [Ninja Bird](https://arcade.makecode.com/#tutorial:github:drb0rn4gr8ness/makecode-tutorials/tutorials/ninjabird)

- [RocketBlaster](https://arcade.makecode.com/#tutorial:github:drb0rn4gr8ness/makecode-tutorials/tutorials/rocketblaster)

- [Star Catcher](https://arcade.makecode.com/#tutorial:github:drb0rn4gr8ness/makecode-tutorials/tutorials/starcatcher)

## 🏫 Classroom Concept Lessons

40-minute Sensei-led lessons for school visits and group settings. Concepts are explicitly taught and assessed with Check on Learning checkpoints.

- [Groovy Star](https://arcade.makecode.com/#tutorial:github:drb0rn4gr8ness/makecode-tutorials/tutorials/groovystar) — Variables, Sequencing, Events, Conditionals, Difficulty Scaling, Power-Ups

- [Meteor Shower](https://arcade.makecode.com/#tutorial:github:drb0rn4gr8ness/makecode-tutorials/tutorials/meteorshower) — Sequencing, Loops

- [Fruit Basket](https://arcade.makecode.com/#tutorial:github:drb0rn4gr8ness/makecode-tutorials/tutorials/fruitbasket) — Arrays, Push, Length, Clearing
